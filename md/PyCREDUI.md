# PyCREDUI

## PyCREDUI_INFO Object

A dictionary representing a CREDUI_INFO structure, used with[win32cred::CredUIPromptForCredentials](win32cred.md#win32credcreduipromptforcredentials)

#### Comments
All members are optional
