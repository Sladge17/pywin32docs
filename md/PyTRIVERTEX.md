# PyTRIVERTEX

## PyTRIVERTEX Object



Dict representing a TRIVERTEX struct containing color information at a point

#### Win32 API References

    Y coord in logical units

  - int Red
    Red component

  - int Green
    Green component

  - int Blue
    Blue component

  - int Alpha
    Transparency value
