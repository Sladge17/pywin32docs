# PySSortOrderItem

## PySSortOrderItem Object

An item in a SortOrderSet.

#### Parameters


  -  *propTag* : int

    A property tag.

  -  *order* : int

    The order in which the data is to be sorted. Possible values are: 

mapi.TABLE_SORT_ASCEND, mapi.TABLE_SORT_COMBINE and mapi.TABLE_SORT_DESCEND
